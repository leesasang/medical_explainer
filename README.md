

## 주요 기능
- 검사지는 한 번에 하나만 선택
- 항목 카드형 / 그룹형 UI
- 혈압 한 줄 입력 (예: 120/80)
- 참고치 수정은 펼침(Expander) 안에서만 가능
- 입력 완료율 표시
- 해석 결과 이상치 색상 강조
- 해석 생성과 PDF 변환 버튼 분리
- PDF 리포트 다운로드

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 변경
- 기본 참고치를 국민건강보험공단(NHIS) 및 보건복지부(MOHW) 공개 자료 중심으로 재점검해 반영
- 반영 항목: 공복혈당, HbA1c, 지질, AST/ALT, GGT, 크레아티닌, eGFR, WBC, 혈색소, 혈소판, 소변검사 기본 기준
- 주의: 실제 검진기관 참고치는 검사법과 기관에 따라 다를 수 있으므로, 필요 시 앱의 참고치 수정 기능을 사용하세요.
